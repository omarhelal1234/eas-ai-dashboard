How it works: Toggles control visibility of sidebar sections across the web dashboard and VS Code extension. Enabled (green) = visible, Disabled (red) = hidden. Dashboard changes take effect on the user's next page load; extension changes on the next sidebar refresh. Default: all visible (deny-list approach). Dashboard permissions work in addition to role-based restrictions — admin can hide views but cannot grant access beyond what the role allows.
