Copilot Enablement

Getting started with GitHub Copilot, AI tools, and official training resources at Ejada

Getting Started

GitHub Enterprise Login

Access the Ejada GitHub Enterprise portal to begin your AI-powered development journey.

Open GitHub Enterprise Portal

Copilot Access Types

Developer GitHub Enterprise + GitHub Copilot Business

Full repository access with Copilot code suggestions in IDE (VS Code, IntelliJ, etc.)

Non-Developer GitHub Copilot Business Only

Copilot Chat only — no repository access. For PMs, BAs, Architects, Consultants, Designers, Testers, etc.

How to Get Access

  1. Prepare a list of resources at project level or department level
  2. Raise an IT ticket specifying the correct access type (Developer vs Non-Developer)
  3. Obtain Manager Approval & Mohamed Al Banna's Approval
  4. Once approved, set up your IDE plugin or Copilot Chat

Important Reminders

  • Stop using personally subscribed or Public AI. All AI usage must go through Ejada-approved enterprise tools.
  • Developers: Integrate your IDE (VS Code, IntelliJ, etc.) with the GitHub Copilot Plugin.
  • Non-Developers: Use Microsoft Copilot Chat for everyday AI assistance.

Installation Guide

Step 1: Download / Clone

The extension lives in the vscode-extension/ folder of this repository.

git clone https://github.com/ejada/E-AI-S.git
cd E-AI-S/vscode-extension

Step 2: Install Dependencies & Build

npm install
npm run compile

Step 3: Package as VSIX

npm run package
# Creates eas-task-logger-1.0.0.vsix

Step 4: Install in VS Code

code --install-extension eas-task-logger-1.0.0.vsix

Or: VS Code → Extensions → ⋯ → "Install from VSIX..." → select the file.

Step 5: Sign In & Start Logging

Open Command Palette (Ctrl+Shift+P) → EAS: Sign In → enter your dashboard credentials.

Then try EAS: Quick Log Task or click the EAS icon in the sidebar.
